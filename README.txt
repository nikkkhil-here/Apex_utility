HOW TO DEPLOY
--------------
Upload the entire apex-utility/ folder to any web host:
  • cPanel / Hostinger / GoDaddy  → upload to public_html/
  • Netlify / Vercel               → drag & drop the folder
  • GitHub Pages                   → push to repo
No build step needed. Works out of the box.

TO VIEW LOCALLY
----------------
Open index.html directly in any browser. Internal links work
because all paths are relative.
